# dtbookOxygen

## Description

dtbookOxygen is a Oxygen Xml Editor plugin for DAISY/NISO Z39.86 dtbook xml files - supported versions are v1.1.0 and 2005-3

Oxygen version 14.0 is supported, may also work with earlier versions.
